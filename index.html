更换主题

主题可以在官网进行搜搜索，官网：Hexo Themes，建议使用Top1主题 『Next』。

1.下载主题。主题一般以github仓库的形式存在，需要通过clone或下载，把主题搞到本地；

2.拷贝至themes目录；

3.在_config.yml，更改主题

Github Pages 部署

需要做三件事：

1.创建github pages仓库；

  • 建立同名仓库即可，用户名wangtianya，则建立wangtianya.github.io

2.添加本地配置(_config.yml)

3.生成静态文件，并部署

4.绑定自定义域名

  • 在pages仓库的设置里自定义。
  • 本地source文件夹添加CANME文件，填上自己的域名，防止deploy后远程仓库清除配置。参考 hexo 上传后gitHub 中custom domain 被重置
