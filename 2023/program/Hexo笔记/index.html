主题

更换主题

主题可以在官网进行搜搜索，官网：Hexo Themes，建议使用Top1主题 『Next』。

1.下载主题。主题一般以github仓库的形式存在，需要通过clone或下载，把主题搞到本地；

2.拷贝至themes目录；

3.在_config.yml，更改主题

## Themes: https://hexo.io/themes/
theme: next

主题美化

字体美化

// 正文字体太大，缩小一点点
$font-size-large          = 1em; // 1.125em -> 15em

// base.styl
$grey-lighter = #fff; // #ddd 副标题灰，改为纯白
$black-deep   = #33a1c9; // #222 默认主黑，改为孔雀蓝

圆角美化

# Gemini.styl 自己用的哪个主题，就改哪个
// $border-radius-inner     = initial;
// $border-radius           = initial;
$border-radius-inner     = 0 0 20px 20px;
$border-radius           = 20px;

安装及部署

本地安装、部署

npm install hexo-cli -g # 安装
hexo init blog # 创建仓库
cd blog
npm install # 安装依赖
hexo server # 部署

Docker部署

如果自己有服务器，可以选择用docker快速部署

# docker-compose
version: "3.3"
services:
  ya-api:
    build:
      context: .
      dockerfile: docker.file
    restart: always
    extra_hosts:
      - "host_addr:host-gateway"
    ports:
      - "8081:4000"
    volumes:
      - /share/ya/codes/blog:/workspace
# docker-file
FROM node:18-alpine
WORKDIR /workspace

RUN echo "Asia/Shanghai" > /etc/timezone \
    && echo "https://mirrors.ustc.edu.cn/alpine/v3.9/main/" > /etc/apk/repositories  \
    && npm config set registry https://registry.npm.taobao.org \
    && apk add --no-cache git \
    && npm install hexo-cli -g

ADD *.sh /
RUN chmod 777 /*.sh
ENTRYPOINT ["sh", "/start.sh"]
# start.sh
echo "$(pwd)"
npm install
hexo server

Github Pages 部署

需要做三件事：

1.创建github pages仓库；

  • 建立同名仓库即可，用户名wangtianya，则建立wangtianya.github.io

2.添加本地配置(_config.yml)

# _config.yml 
## Docs: https://hexo.io/docs/one-command-deployment
deploy:
  type: git
  repository: https://github.com/wangtianya/wangtianya.github.io
  branch: master

3.生成静态文件，并部署

$ hexo clean && hexo generate && hexo deploy # 部署至github

4.绑定自定义域名

  • 在pages仓库的设置里自定义。
  • 本地source文件夹添加CANME文件，填上自己的域名，防止deploy后远程仓库清除配置。
